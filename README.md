# Vacation Planner Mobile App

This repository hosts the deployment page for the **Vacation Planner** Android application developed for the **WGU Software Engineering Capstone (D424)**.

The application allows users to plan vacations, manage excursions, receive date alerts, and generate reports of their travel plans.

## Download the Application

You can download and install the Android APK here:

➡️ https://ChrisF-042.github.io/vacation-planner-app/

Once downloaded, install the APK on an Android device or emulator.

## Features

- Create and manage vacations
- Add and track excursions for each vacation
- Search vacations with dynamic filtering
- Generate vacation summary reports
- Share vacation details
- Date alerts for vacation start and end

## Technology Stack

- **Java**
- **Android Studio**
- **Room ORM (SQLite)**
- **RecyclerView**
- **GitLab (version control)**
- **GitHub Pages (deployment)**

## Project Purpose

This application was developed to address the organizational need for a simple mobile tool that helps users track vacation plans, excursions, and important travel dates in one place.
